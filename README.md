Denny's vimrc
===

How to use?
---

backup your origin `.vimrc` and `.vim` settings.

in home directory

```
git clone https://github.com/denny0223/.vim.git && ln -s .vim/.vimrc
```

use vim open any file, vim plugin will auto install.
